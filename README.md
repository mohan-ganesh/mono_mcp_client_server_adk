In this repo, you have 4 modules. 


`mcp_billing_server` - MCP Billing Server module that hosts billing related tools.

`mcp_email_server` - MCP Email Server module that hosts email related tools.

`mcp_cli_client` - MCP Command Line Interface Client module that connects to MCP Server's.

`mcp_orchestrator_client` - Spring Boot based MCP Orchestrator Client module that connects to MCP Server's.
