McpCliClient is a command-line interface (CLI) application that interacts with a Firestore database to manage and retrieve data related to users, products, and orders.

`mvn clean compile exec:java -Dexec.mainClass="com.example.garvik.McpCliClient"`

McpCliInMemoryClient is a command-line interface (CLI) application that interacts with an in-memory database to manage and retrieve data related to users, products, and orders.

`mvn clean compile exec:java -Dexec.mainClass="com.example.garvik.McpCliInMemoryClient"`
